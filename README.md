## Tuto lancement du compilateur

Utiliser la suite de commande suivante pour utiliser le compilateur sur le fichier de votre choix:

```bash
make parser
```

```bash
make compilateur
```

```bash
make run target
```
où target et le chemin vers votre fichier cible
